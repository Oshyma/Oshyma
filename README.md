## Featured Projects


### [MeetingSync](https://github.com/Oshyma/meetingsync)
A micro-SaaS for automatic meeting transcription and summarization — upload recordings and receive structured email summaries with key points, decisions, and action items. Built with scalable processing pipelines, authentication, payments, and quota-based usage management.
`Next.js 14` `TypeScript` `Supabase` `OpenAI Whisper` `GPT-4o` `Stripe` `Resend`

### [PrometheaBot](https://github.com/PrometheaCorp/PrometheaBot)
A full-featured Discord bot for community management — XP leveling, anti-bot captcha, music playback (YouTube / Spotify / SoundCloud), moderation, audit logging, and server setup utilities.
`Discord.js v14` `DisTube` `SQLite` `Node.js`

---

### [Promethea Delivery](https://github.com/PrometheaCorp/Promethea-Delivery)
A Discord bot that handles end-to-end order management — delivery tickets, a complete affiliate program with unique codes and PayPal tracking, HTML transcripts, and cross-server order verification.
`Discord.js v14` `quick.db` `SQLite` `Node.js`

---

### [AscendOs](https://github.com/Oshyma/AscendOs) *(WIP — on hold)*
A VS Code extension that turns coding into an RPG. Earn XP as you write code, level up, complete missions, and collect loot — all from your editor's status bar.
`TypeScript` `VS Code API` `ESBuild`
